Release Changes

The following features have changed or improved.

  • Core Modules
    • The sfb-cli application now depends on ASK CLI v2. Follow the instructions at Quick Start: Alexa Skills Kit Command Line Interface (ASK CLI) to download the latest version of ASK CLI. If you do not upgrade your ASK CLI to 2.0.0 or later, sfb-cli will automatically download and run the latest version of ASK CLI when needed for deployment.
    • By default, new projects created with sfb-cli will use ASK CLI's "cfn" deployer to deploy resources with AWS Cloud Formation. If you wish to use the "lambda" deployer instead, include the --deployer lambda option when running the alexa-sfb build or alexa-sfb deploy commands.
      • Note: You cannot change the deployer once the skill has been deployed, and existing projects will automatically use the "lambda" deployer when upgrading to sfb-cli 2.0.0 or higher.
    • You can now deploy the node_modules directory under your SFB project's code directory as an AWS Lambda layer, which will only get redeployed if the node_modules directory changes. To enable using Lambda layer, set "use-lambda-layer" to true in your projects abcConfig.json file.
